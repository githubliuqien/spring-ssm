2.动态SQL
    <sql>       进行 SQL 语句的封装，一般用于字段名封装，通过 <include> 加载封装的 SQL 语句
    <set>       替代 set 关键词，会自动省略后缀","
    <if>        判断赋予给字段的参数是否为 null，为 null 则不添加,不为 null 则进行填充
    <where>     替代 where 关键词，会自动省略 where 后面的 and 或者 or
    <trim>      自定义标签，可以添加前缀和后缀，自定义根据需要删除前缀和后缀的关键词
                 prefix：可选属性，用于指定前缀字符串。如果 SQL 语句片段的前缀与该属性匹配，则该前缀会被移除。
                 suffix：可选属性，用于指定后缀字符串。如果 SQL 语句片段的后缀与该属性匹配，则该后缀会被移除。
                 prefixOverrides：可选属性，用于指定要移除的开始部分。只有当 SQL 语句片段以指定的某个字符串为开始时，才会被修剪。
                 suffixOverrides：可选属性，用于指定要移除的结束部分。只有当 SQL 语句片段以指定的某个字符串为结束时，才会被修剪。

    <foreach>   遍历集合，collection 属性为要遍历的集合，
                        open 属性为开始遍历前添加的内容，
                        separator 属性为循环添加的内容
                        close 属性为遍历结束后添加的内容
                        item 属性为循环值
